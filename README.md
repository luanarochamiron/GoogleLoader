# Google Loader Animation using HTML & CSS

Um componente de indicador de carregamento (Loader/Spinner) moderno, dinâmico e de alta performance desenvolvido exclusivamente com HTML5 e CSS3. O projeto recria o clássico carregador da identidade visual do Google, utilizando quatro esferas coloridas que realizam um movimento de salto ondulatório sincronizado através de atrasos de animação.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica simples do contêiner do carregador e das quatro esferas independentes.
* CSS3: Estilização completa, formas circulares perfeitas, aplicação da paleta cromática oficial e gerenciamento de sombras.
* Animações em CSS: Utilização de keyframes (`@keyframes`) avançados combinados com a propriedade de atraso (`animation-delay`) para criar o efeito de onda.

## Funcionalidades e Técnicas Aplicadas

* Paleta de Cores Oficial: Renderização precisa das esferas utilizando as cores de marca integradas: azul, vermelho, amarelo e verde.
* Animação Wave/Bounce Fluida: Movimento vertical cíclico e contínuo onde cada elemento salta em tempos ligeiramente diferentes, gerando um efeito visual harmônico de onda elástica.
* Sincronização Otimizada: Uso estratégico da propriedade `animation-delay` com valores fracionados em segundos para coordenar o ritmo exato de subida e descida de cada círculo.
* Efeito de Profundidade Discreto: Aplicação de uma sombra projetada (`box-shadow`) muito suave abaixo de cada esfera, conferindo uma sensação de flutuação e tridimensionalidade em relação ao fundo claro.
* Centralização Absoluta com Flexbox: Alinhamento horizontal simétrico das esferas e posicionamento centralizado no eixo x/y da viewport.
